# 欢乐斗牛 · 联机版（同一 WiFi）

手机联机斗牛，最多 8 人，人不够 AI 补齐。

## 一、启动服务器（房主的电脑，只需一次）

1. 确保装了 [Node.js](https://nodejs.org)（你电脑已装 v24）。
2. 打开终端（PowerShell），进入本文件夹：
   ```
   cd "C:\Users\19775\Desktop\新建文件夹 (2)\doudou-online"
   ```
3. 第一次先装依赖（之后不用再装）：
   ```
   npm install
   ```
4. 启动：
   ```
   npm start
   ```
   看到这样的输出就成功了：
   ```
   本机访问：  http://localhost:3000
   手机访问：  http://192.168.x.x:3000   （手机连同一 WiFi）
   ```

> 端口被占用？用 `set PORT=3001 && npm start`（PowerShell：`$env:PORT=3001; npm start`）。

## 二、房主开房

- 电脑或手机浏览器打开上面的地址 → 输昵称 → **创建房间** → 得到 **4 位房间号**和二维码。
- （可选）用 **＋/－** 设定总人数（2~8），空位自动 AI 补齐 → **开始游戏**。

## 三、朋友加入（关键）

1. 朋友手机**连同一个 WiFi**（和你电脑同一个路由器 / 或连你电脑开的热点）。
2. 手机**扫房间页的二维码**，或浏览器输入 `http://你的IP:3000`，再填房间号。
3. 房主点「开始游戏」即可。

## 四、连不上？（Windows 防火墙）

第一次运行时如果弹出「是否允许 Node.js 通信」，**勾选「专用网络」并允许**。
如果没弹窗又连不上，手动放行：控制面板 → Windows Defender 防火墙 → 允许应用 → 找到 Node.js 勾选「专用」。

多个 `192.168 / 172` 地址时，用**和手机同一网段**的那个（一般是 `192.168.x.x`）。

## 玩法

发牌 → 抢庄（倍数高者坐庄）→ **道具阶段**（🔄换牌 / ✖️2翻倍，每局限一个）→ 下注 → 亮牌 → 庄闲结算。
牌型：五小牛 > 五花牛 > 牛牛 > 牛9…牛1 > 没牛。赢一局送 1 张随机道具。

> 单机离线版仍在上一层文件夹的 `斗牛.html`，双击即玩。
