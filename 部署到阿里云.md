# 欢乐斗牛部署到阿里云

适合长期和朋友公网联机。推荐买阿里云“轻量应用服务器”或 ECS，系统选 Ubuntu 22.04 / 24.04。

## 服务器建议

- 1 核 1G 可以跑这个小游戏。
- 地域选离你和朋友近的，比如华东、华南。
- 安全组/防火墙需要开放 TCP `3000` 端口。

## 1. 登录服务器

在阿里云控制台找到公网 IP，然后用 SSH 登录：

```bash
ssh root@你的服务器公网IP
```

## 2. 安装 Node.js 和 Git

```bash
apt update
apt install -y git curl
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs
node -v
npm -v
```

## 3. 拉取项目

```bash
cd /opt
git clone https://github.com/ZhangShuHaoao/douniu.git
cd douniu
npm ci
```

## 4. 先测试运行

```bash
npm start
```

看到“欢乐斗牛 · 联机版已启动”后，在浏览器访问：

```text
http://你的服务器公网IP:3000
```

确认能打开后，按 `Ctrl+C` 停止临时运行。

## 5. 用 PM2 常驻运行

```bash
npm install -g pm2
pm2 start ecosystem.config.cjs
pm2 save
pm2 startup
```

`pm2 startup` 会输出一条命令，复制它再执行一次。这样服务器重启后游戏也会自动启动。

常用管理命令：

```bash
pm2 status
pm2 logs douniu
pm2 restart douniu
pm2 stop douniu
```

## 6. 阿里云安全组

如果浏览器打不开，优先检查安全组：

- 入方向
- 协议类型：TCP
- 端口范围：3000/3000
- 授权对象：0.0.0.0/0

然后访问：

```text
http://你的服务器公网IP:3000
```

## 注意

- 房间数据在内存里，重启服务后房间会消失。
- 没有域名时，用公网 IP 加端口访问即可。
- 如果以后绑定域名，可以用 Nginx 反向代理到 `127.0.0.1:3000`。
